A Multisite, bug tracking plugin. Allow you to integrate BugHerd sidebar into WordPress based website/blog within one minute.

== Description ==

BugHerd is a new, modern and sleek bug tracker designed specifically to help web designers and developers to visually describe the bug(s) in a layout, plugin also provide some basic backend administration to allow you to select different user levels to report a bug from visitors to administrators and to create/select projects within WordPress administration.


<h4>Signing up</h4>

Signing up with <a href="http://bugherd.com">BugHerd</a> is one click process and free, plugin required you to have an account with BugHerd and if you have multiple blogs we recommend you to enable Restful API in your <a href="https://www.bugherd.com/settings">profile settings page</a> and use API feature to login into BugHerd, select and add new project from within your WordPress administration.

<h4>More Information</h4>

Visit the <a href="http://vimeo.com/39621566">BugHerd video demonstration</a> to understand how easily you can assign and track your bugs.
Visit the <a href="http://simpleux.co.uk/plugins/wordpress/bugherd">BugHerd plugin page</a> to see step by step guide to install the plugin.

== Installation ==

You can download and install BugHerd using the built in WordPress plugin installer. If you download manually, make sure it is uploaded to "/wp-content/plugins/bugherd/".

Activate BugHerd in the "Plugins" admin panel using the "Activate" link.

If you are using WordPress Multisite then plugin will allow you to copy the BugHerd settings from main site to subsites automatically or alternatively you can define individual settings in each site.
