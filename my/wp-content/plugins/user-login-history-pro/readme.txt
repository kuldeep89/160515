Displays a history of user logins.

== Description ==
= A simple, easy to use plugin for viewing user logins =
User Login History - Pro displays a clean, simple, and sortable history of your user's logins. This allows you to see exactly who has logging into your site and when.

= Features Include =
*	Sorting
*	Searching
*	Date range selection
*	Filtering per user and per login
* 	Pagination
*	Viewing name, email, last log in, date, and total logins

= To use this plugin =
Using this plugin is easy! All you have to do is upload and activate this plugin. After successful activation, a sub-menu item titled 'User Login History' will appear under the 'Users' menu item. User Login History - Pro was built to be lightweight and easy to use. Enjoy!

Requires WordPress 3.0 and PHP 5.

== Installation ==
I you have User Login History - Free installed, you must remove it before activating the pro version.

= Using The WordPress Dashboard =
1. Navigate to the 'Add New' in the plugins dashboard
2. Search for ‘User Login History - Pro’
3. Click 'Install Now'

= Activate the plugin on the Plugin dashboard =
1. Uploading in WordPress Dashboard
2. Navigate to the 'Add New' in the plugins dashboard
3. Navigate to the 'Upload' area
4. Select user-login-history-pro.zip from your computer
5. Click 'Install Now'
6. Activate the plugin in the Plugin dashboard

= Using FTP =
1. Download user-login-history-pro.zip,
2. Extract the user-login-history-pro directory to your computer,
3. Upload the user-login-history-pro directory to the /wp-content/plugins/ directory,
4. Activate the plugin in the Plugin dashboard,
